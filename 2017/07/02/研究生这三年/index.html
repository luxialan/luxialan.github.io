研究生这三年

Jul 2, 2017 | 随便侃侃总结 | 阅读
0 条参与

谢谢

白煦过隙，研究生三年转眼就过了，很感谢这三年的时光，至今仍记得当年大四时抱着试一试的态度联系现在导师的忐忑心情。这三年伴随实验室一起成长，从最初的十几人扩展到现在的几十人，期间经历了不少事情，回头看由于自己的不成熟当时处理得并不是那么得当，但很幸运地，周围的人大多给予了包容和理解。遇见了值得珍惜的伙伴，对我而言是很重要的存在。同门师兄弟都有各自闪光的地方，所以时常提醒自己要多多努力，身处一个向上发展的团队中，很容易将平台的光环误以为是自身的能力。
